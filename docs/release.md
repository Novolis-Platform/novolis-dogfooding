# Release

This repository does **not** ship packages or installers.

- No version tags for NuGet
- No `release.yml` or trusted publishing workflow
- No CI in this repo; library repos publish to GitHub Packages on merge

Library releases happen in each `novolis-*` library repository per [novolis-governance release policy](https://github.com/Novolis-Platform/novolis-governance/blob/main/docs/release-policy.md).
